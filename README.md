
1. Цель проекта
Создать веб‑ресурс в формате википедии по кастомной карте «Гарена» для Warcraft 3, дополненный функционалом редактора персонажа с возможностью анализа прогресса (уровни, способности, предметы).

2. Целевая аудитория
игроки в «Гарену»;

новички, изучающие механики карты;

аналитики и стримеры, собирающие статистику;

создатели контента (гайды, обзоры).

3. Функциональные требования
3.1. Вики‑раздел

База данных: структурированные статьи по категориям:

герои (характеристики, способности, роли);

предметы (эффекты, стоимость, рецепты);

способности (описание, уровни прокачки, кулдауны);

игровые механики (фарм, линии, нейтралы);

режимы игры (1v1, 2v2, FFA и т. д.);

термины и сленг.

Поиск и навигация:

полнотекстовый поиск;

фильтрация по категориям;

теги для статей;

карта сайта и оглавление.

Редактирование контента:

регистрация пользователей;

система прав (читатели, редакторы, модераторы);

история изменений (версиями);

предпросмотр перед публикацией.

3.2. Редактор персонажа

Создание профиля:

выбор героя из списка;

ввод имени персонажа;

настройка начальных параметров (опционально).

Отслеживание прогресса:

фиксация времени получения уровня (формат: ЧЧ:ММ:СС);

запись прокачанной способности при повышении уровня;

добавление купленных предметов с указанием времени покупки;

возможность отмечать ключевые события (убийства, ассисты, разрушения зданий).

Визуализация данных:

график уровней по времени;

диаграмма распределения очков способностей;

таймлайн покупок предметов;

статистика: среднее время до уровня, частота использования способностей.

Экспорт и импорт:

сохранение сессии в файл (JSON/CSV);

загрузка ранее сохранённых данных для анализа.
Запуск и поддержка (постоянно):

развёртывание на сервере;

мониторинг ошибок;

сбор обратной связи от пользователей
